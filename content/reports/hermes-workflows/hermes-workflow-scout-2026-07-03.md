# Hermes workflow items worth opening today

- 🔥 Best: Instar is closest to a Hermes-adjacent pattern: persistent Claude Code agents with scheduling, sessions, memory, Telegram/WhatsApp, MCP, skills, and agent identity. https://github.com/JKHeadley/instar
- Claw Skeleton is a concrete Telegram personal-assistant skeleton where every topic maps to an isolated Claude Code agent with layered memory and proactive cron heartbeats. https://github.com/LuKresXD/claw-skeleton
- Scarf is a native macOS/iOS client for Hermes itself with multi-window chat, local/SSH servers, sessions, memory, cron, MCP, and dashboard surfaces worth studying for UX ideas. https://github.com/awizemann/scarf
- Context Infrastructure packages persistent coding-agent memory, personal rules, skills, and scheduled observations into a reusable context layer for autonomous coding workflows. https://github.com/grapeot/context-infrastructure
- Agent-Blackbox is a local-first flight recorder that replays coding-agent runs as session maps, scores context usage, and can write fixes back into AGENTS.md. https://github.com/TaewoooPark/Agent-Blackbox
- AgentGraphed is a local-first analytics dashboard for Claude Code and Codex CLI sessions that could inspire Hermes session-insights and report-log views. https://github.com/sudomichael/agentgraphed
- Agent Andon is a tiny self-hosted status board and notification layer for Claude Code/Codex that focuses on “needs input, finished, or stuck” signals for unattended agents. https://github.com/tianshanghong/agent-andon
- Botsman is a Telegram-native AI DevOps agent that turns chat prompts into deployed web services on a user-owned VPS, useful as a reference for phone-first agent ops. https://github.com/alzaxnsk-del/botsman
- Friday Studio is a self-hosted agent harness with shareable workspaces, MCP tools, skills, memory, cron, and webhook automations that overlaps strongly with persistent Hermes workflow infrastructure. https://github.com/friday-platform/friday-studio
- Doberman Core targets agent security with guardrails, prompt-injection defense, runtime policy enforcement, tool permissions, monitoring, and audit logs for autonomous workflows. https://github.com/fu351/Doberman-Core
