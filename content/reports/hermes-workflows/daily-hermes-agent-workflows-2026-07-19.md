# Hermes/agent workflows worth opening today

- 🔥 Best: Hermes Atlas is a live, community-curated directory of Hermes Agent skills, tools, integrations, ecosystem repos, and a RAG assistant for finding useful setups. Source: https://github.com/ksimback/hermes-ecosystem
- Talon is a multi-platform agent harness that can run through Telegram, Discord, Teams, Terminal, and mobile/desktop while connecting Claude, OpenCode, Codex, OpenAI Agents, MCP tools, and persistent background agents. Source: https://github.com/dylanneve1/talon
- Luma Assistant is a self-hosted browser workspace for Codex and Claude Code with cron jobs, sandbox terminals, offline voice-to-text, MCP tools, skills, persistent sessions, and Telegram automation. Source: https://github.com/mertz1999/luma-assistant
- OtoDock lets a team run Claude Code and Codex agents together on a server with shared dashboards, security boundaries, and company workflows. Source: https://github.com/OtoDock/oto-dock
- Fluxito is an MCP-ready marketing analytics platform that connects AI agents to many analytics, tag-manager, dashboard, audit, and reporting workflows from one conversation. Source: https://github.com/digitalXperiments/fluxito
- Octopus LinkedIn is a governed LinkedIn MCP server and CLI for drafting, reviewing, approving, publishing, commenting, and AI-writing through the official LinkedIn API. Source: https://github.com/octoryn/octopus-linkedin
- Echo Agent is a self-hosted long-running AI agent with layered memory, skill evolution, approvals, audit logs, MCP/A2A support, cron, webhooks, Telegram, Slack, Discord, WeChat, and other channels. Source: https://github.com/fuyuxiang/echo-agent
- ForgeDock turns GitHub issues and milestones into durable autonomous coding pipelines with scheduling, recovery, review, memory, and PR-oriented execution. Source: https://github.com/RapierCraftStudios/ForgeDock
- Pi Digby is a self-hosted Slack and Linear agent with persistent memory files, shell/file tools, MCP integrations, and scheduled events that can be adapted as a small-team operator pattern. Source: https://github.com/brainwavesio/pi-digby
- OpenLoomi is a local-first open-source AI coworker focused on desktop attention, work memory, and builder workflows around personal agents. Source: https://github.com/melandlabs/openloomi
