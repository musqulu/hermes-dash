# Hermes workflow items worth opening today

- 🔥 Best: OtoDock is a self-hosted server/dashboard for running Claude Code and Codex as coordinated company agents, directly relevant to Hermes multi-agent/profile orchestration ideas: https://github.com/OtoDock/oto-dock/
- Session Briefing is a lightweight continuity pattern that separates stable AGENTS.md context from timestamped rolling project state and cross-project rollups: https://github.com/CaptCanadaMan/session-briefing
- Echo Agent is a self-hosted long-running agent with multi-channel gateways, cron/webhooks, four-layer memory, skill evolution, approvals, and auditability: https://github.com/fuyuxiang/echo-agent
- Claude Code Config is a practical rules/hooks/skills kit for agentic development workflows, including verification discipline, context management, and multi-agent operating patterns: https://github.com/AnastasiyaW/claude-code-config
- HeyClaude is a file-backed registry of Claude agents, MCP servers, skills, hooks, commands, tools, prompts, rules, and guides that could inspire Hermes skill/tool catalogs: https://github.com/JSONbored/awesome-claude
- AI Agent Observer provides SDK and MCP-based tracing for multi-agent runs with spans, model tiers, token cost, and failure points in one dashboard: https://github.com/gusrodriguez/ai-agent-observer
- Censai is a self-hosted local-first canvas for persistent agents with memory, scheduled background workers, sub-agents, and real-time file/code/document work: https://github.com/oogalieboogalie/censai
- Vendo shows a useful embedded-agent pattern where a product agent can build views, call guarded APIs, and run an install/doctor loop inside an app: https://github.com/runvendo/vendo
- Libretto debug agents are a focused workflow for automatically repairing failing Playwright scripts, useful as a concrete pattern for Hermes test-failure remediation agents: https://libretto.sh/debug-agents
- Pokayoke turns code conventions into checks for agents, which maps well to Hermes preflight hooks or project-specific guardrail skills: https://pokayoke.codes
