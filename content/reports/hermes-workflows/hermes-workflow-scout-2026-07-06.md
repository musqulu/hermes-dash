# Hermes workflow items worth opening today

- 🔥 Best: Aharness enforces coding-agent workflows as typed state machines with gates, validated evidence, repair paths, and inspectable logs for long Codex runs: https://github.com/Alfredvc/aharness
- Stratum packages governed agent workflows as an MCP server and Python library with typed YAML specs, postconditions, retries, gates, and auditable execution traces: https://github.com/smartmemory/stratum
- Metidos is a local developer workspace for managing AI coding agents across projects, worktrees, threads, diffs, cron jobs, permissions, and handoff-friendly context: https://github.com/jtenner/metidos
- Session Briefing separates stable AGENTS.md-style context from timestamped current-state rollups so coding agents can resume with fresher project memory: https://github.com/CaptCanadaMan/session-briefing
- Another Orchestrator uses a planner/runner split where JSON plans feed a deterministic YAML phase graph that dispatches Claude Code or Codex in isolated git worktrees: https://github.com/linuxlewis/another-orchestrator
- Hamroh is a self-owned Telegram persistent companion harness with memory, scheduled tasks, MCP/tool extension points, logs, and a custom-agent template: https://github.com/Rustam-Z/hamroh
- Semantic Daily is a multi-source agentic digest pipeline that collects Reddit, Bluesky, GitHub, RSS, Hacker News, YouTube, and Exa signals, deduplicates them, and delivers Telegram or dashboard briefs: https://github.com/ashok-kallara/semantic-daily
- Long-Running Tasks is an OpenClaw skill pattern for TODO.md-driven multi-phase coding agents with cron orchestration, stall detection, and automatic respawn rules: https://github.com/mmTheBest/long-running-tasks
