# Hermes workflow items worth opening today

- 🔥 Best: TokenTelemetry gives Hermes/Claude/Codex a local dashboard for tokens, sessions, tool calls, and reasoning, making it the most immediately useful observability idea to copy. https://github.com/VasiHemanth/tokentelemetry
- ShibaClaw is a security-first self-hosted agent with 11 chat channels, 3-level memory, task scheduling, automation, skills, and MCP that is worth mining for gateway and safety patterns. https://github.com/RikyZ90/ShibaClaw
- Echo Agent is a long-running self-hosted agent with layered memory, forgetting/contradiction handling, LLM-based approvals, MCP/A2A, scheduling, and Telegram/Gateway entry points. https://github.com/fuyuxiang/echo-agent
- AstrBot is a mature multi-platform IM agent framework with plugins and many chat adapters that can inform Hermes gateway ergonomics and platform abstractions. https://github.com/AstrBotDevs/AstrBot
- Mempalace Evolve focuses on self-improving persistent memory with automatic learning, a knowledge graph, and multi-agent support, useful for thinking about memory rollups. https://github.com/a2328275243/mempalace-evolve
- Tandem is an authority layer for agent tool access, memory, approvals, and audit trails, directly relevant to safer proactive Hermes workflows. https://github.com/frumu-ai/tandem
- Entire CLI records AI-agent sessions alongside Git commits, a strong pattern for making autonomous coding work searchable and auditable after the fact. https://github.com/entireio/cli
- Dagu is a local-first workflow engine with declarative YAML DAGs and a web UI that could complement or inspire more inspectable Hermes cron/proactive workflows. https://github.com/dagucloud/dagu
- pi-agent-dashboard provides a real-time multi-session dashboard, live chat mirroring, terminal, diff viewer, and mobile remote control for coding-agent sessions. https://github.com/BlackBeltTechnology/pi-agent-dashboard
- claude-code-config collects Claude Code, Codex, hooks, skills, and multi-agent workflow patterns that are directly reusable as reference material for Hermes skill/workflow design. https://github.com/AnastasiyaW/claude-code-config
