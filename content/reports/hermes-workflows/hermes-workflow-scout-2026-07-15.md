# Hermes workflow items worth opening today

- 🔥 Best: Dapr Agents is directly relevant to Hermes because it combines stateful agents, durable workflow orchestration, telemetry, and security boundaries in one Python framework: https://github.com/dapr/dapr-agents
- Echo Agent is a self-hosted long-running personal/team agent with four-layer memory, forgetting/contradiction checks, MCP/A2A, scheduling, approvals, and Telegram/WeChat gateways worth mining for persistent-agent patterns: https://github.com/fuyuxiang/echo-agent
- Agent Blackbox is a local-first flight recorder for coding agents that maps runs, scores context cost, and writes lessons back into AGENTS.md-style guidance: https://github.com/TaewoooPark/Agent-Blackbox
- JAT is an agentic IDE/dashboard for supervising many coding-agent sessions with task management, terminals, auto-proceed rules, and swarm workflows: https://github.com/joewinke/jat
- Fleet Deck is a small board for viewing every local Claude Code session, useful as a concrete reference for a Hermes multi-session dashboard: https://github.com/lacion/fleet-deck
- Leash watches overnight agent work for costs, loops, dead crons, and failing workflows, matching the kind of watchdog Hermes cron/proactive runs need: https://github.com/VicKayro/leash
- Cronicle describes AI-agent and shell tasks as declarative HCL cron/DAG workflows with budget caps and per-run transcripts: https://github.com/jshiv/cronicle
- Coding Agent Skill Library packages reusable coding-agent skills as MCP-served assets, a useful model for distributing Hermes skills beyond local Markdown files: https://github.com/mmccalla/coding-agent-skill-library
- Doberman Core focuses on runtime policy enforcement, tool-use permissions, monitoring, and audit logs for autonomous agents, which maps well to Hermes approval/security hardening: https://github.com/fu351/Doberman-Core
- AstrBot is a mature multi-platform IM agent framework with plugins and many messaging adapters, useful for comparing Telegram/Discord gateway ergonomics and plugin architecture: https://github.com/AstrBotDevs/AstrBot
