# Hermes/agent workflows worth opening today

- 🔥 Best: SuearchHermes adds free Google Search to Hermes Agent via Gemini grounding, useful for replacing paid search backends in research cron jobs. https://github.com/leonardo-ferioli/SuearchHermes
- Caliper is a lightweight eval harness for Hermes, Codex, Claude Code, and Pi skills, useful before trusting custom operator workflows. https://github.com/edonadei/caliper
- telegram-notifier-mcp gives any MCP-compatible agent one clean Telegram send tool for “tell me when the job finishes” workflows. https://github.com/dtiger1889-ops/telegram-notifier-mcp
- telegram-agent shows a serverless Telegram personal assistant on Cloud Run with GitHub-backed memory and swappable LLM providers. https://github.com/cherrythia/telegram-agent
- Agentled MCP packages workflow orchestration, long-term memory, and 100+ integrations behind one agent-accessible automation server. https://github.com/Agentled/mcp-server
- n24q02m’s Claude plugin marketplace bundles installable MCP tools for web search, memory, Notion, email, Telegram, code review, and media workflows. https://github.com/n24q02m/claude-plugins
- Dex turns an agent into an analytics-engineering helper for dbt, warehouses, schema drift, and repeatable data-stack maintenance. https://github.com/exmergo/dex
- after-effects-mcp lets Codex, Cursor, or Claude Code drive Adobe After Effects through MCP tools for content and motion-graphics automation. https://github.com/JUNKDOGE-JOE/after-effects-mcp
- Project NOVA is a multi-agent n8n plus MCP architecture with prompts, Dockerfiles, and workflows for routing work to specialized agents. https://github.com/PradeepaRW/project-nova
- Rowboat is an open-source desktop AI coworker with memory and built-in action surfaces, worth studying for personal-operator UX patterns. https://github.com/rowboatlabs/rowboat
