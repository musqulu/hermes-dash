# Hermes workflow items worth opening today

- 🔥 Best: Lucarne is a practical “stop babysitting agents” pattern for push notifications plus approve/resume loops across Codex, Claude Code and similar local coding agents. https://github.com/tuchg/Lucarne
- agent-reachout shows a focused Telegram decision-request workflow for Claude Code that maps cleanly to Hermes gateway approvals and proactive agent nudges. https://github.com/vibe-with-me-tools/agent-reachout
- SwarmClaw is a self-hosted UI for coordinating OpenClaw agent swarms, useful as a reference for Hermes multi-profile dashboards and worker visibility. https://github.com/swarmclawai/swarmclaw
- Kodama is a self-hosted autonomous daemon for Claude Code and Codex that is worth mining for durable background-agent lifecycle patterns. https://github.com/FratteFlorian/kodama
- TokenTelemetry is an observability dashboard for autonomous/coding agents that tracks sessions, tokens and tool calls, directly relevant to Hermes insights and dashboard UX. https://github.com/VasiHemanth/tokentelemetry
- Droidctx generates Markdown context from production infrastructure for coding agents, a useful pattern for cron-built context packs before autonomous code work. https://github.com/DrDroidLab/context-builder
- kmux is a terminal workspace for running parallel Claude Code, Codex CLI and other agents, useful for Hermes tmux/worktree orchestration ergonomics. https://github.com/kkd927/kmux
- agent-control turns CLI coding agents into a job-board style autonomous workforce, a compact reference for task claiming, status and review loops. https://github.com/serifcolakel/agent-control
- superharness experiments with a multi-agent orchestration harness for Claude Code plus Codex with task lifecycle, handoffs and a dashboard. https://github.com/artificemachine/superharness
