# Hermes workflow items worth opening today

- 🔥 Best: OtoDock is a self-hosted platform for running real Claude Code and Codex agents as a collaborative team with dashboards, schedules, triggers, meetings, memory, documents, voice, and sandbox controls. https://github.com/OtoDock/oto-dock
- JAT packages an agentic IDE around live sessions, task management, code editor, terminal, external-source triggers, auto-proceed rules, and supervision of 20+ coding agents. https://github.com/joewinke/jat
- Better Telegram MCP exposes Telegram chats, messages, media, and contacts to agents across both bot and full user-account modes, which is directly relevant to richer Hermes Telegram workflows. https://github.com/n24q02m/better-telegram-mcp
- aharness turns long-running Codex workflows into typed finite-state machines with validated gates, evidence receipts, repair paths, model-per-state selection, and inspectable logs. https://github.com/Alfredvc/aharness
- cat-harness is a Claude Code plugin that auto-routes vague implementation prompts through interview, plan approval, evidence-gated execution, and optional team parallelization. https://github.com/chussum/cat-harness
- Agent HQ is a zero-dependency MCP platform for agent companies with shared memory, kanban, agent registry, live dashboard, WIP limits, spend tracking, and SSE updates. https://github.com/tools-for-agents/agent-hq
- boardstate-hermes-plugin is a Hermes dashboard plugin prototype that lets an agent compose live dashboard boards via MCP-backed Boardstate widgets bound to Hermes usage, sessions, status, and cron data. https://github.com/100yenadmin/boardstate-hermes-plugin
- Orchestra combines persistent Claude CLI sessions, worktree-isolated workers, GPT review, a FastAPI dashboard, and a Telegram bridge into a manager-style multi-agent orchestration pattern. https://github.com/DrSeedon/orchestra
- trusty-tools bundles Rust MCP servers for hybrid code search and long-term semantic memory, offering a practical reference for machine-wide code indexes plus memory-palace retrieval. https://github.com/bobmatnyc/trusty-tools
- agentic-awesome-skills is a very large installable catalog of reusable SKILL.md playbooks and bundles across Claude Code, Cursor, Codex CLI, Gemini CLI, and related agent tooling. https://github.com/sickn33/agentic-awesome-skills
