# Hermes workflow items worth opening today

- 🔥 Best: FastClaw is a compact reference for turning a chat subscription into a Slack/Telegram employee with Rust server, SQLite queue, guardrails, cron, self-improving memory, and an admin dashboard: https://github.com/Grail-Computer/FastClaw
- Agenhood is a self-hosted 24/7 autonomous-agent platform with Docker sandboxes, persistent memory/workspaces, scheduling, workflows, live streaming, and pluggable Claude Code/Codex/opencode brains: https://github.com/appssemble/agenhood
- Aharness shows how to enforce coding-agent workflows as typed state machines with gates, evidence validation, repair paths, controlled transitions, and inspectable logs: https://github.com/Alfredvc/aharness
- The Vault is a local-first memory layer with desktop app, CLI, and MCP server for durable project context, handoffs, recall, and task continuity across coding agents: https://github.com/aliihsaad/the-vault
- Relaymux is a lightweight tmux-backed meta-harness for supervising and relaying between local coding agents, useful for Hermes multi-agent terminal orchestration ideas: https://github.com/mupt-ai/relaymux
- AutomatiQ is a browser-automation assistant that reverse-engineers web apps into reusable scraping/automation actions, relevant for building safer source collectors for scouts: https://github.com/StoneSteel27/AutomatiQ
- Aegize frames the missing runtime layer for autonomous agents—identity, policy, permissions, approvals, audit logging, observability, and runtime governance around every tool action: https://www.aegize.com/
- Sol proposes validated computational artifacts for human-agent handoffs, a useful pattern for making Hermes outputs verifiable rather than just conversational: https://github.com/LopezNuance/Sol
- StageWhisper Lite is an on-device meeting/call note layer that can pair with OpenClaw/Hermes-style agents as a private ambient context source: https://stagewhisper.io/lite
- Quickchat’s Telegram group-agent guide is a practical Bot API walkthrough for announce, pin, and moderation workflows that maps well to Telegram-first Hermes operators: https://quickchat.ai/post/connect-ai-agent-to-telegram-bot-api
