- Hermes Deck is a native macOS multi-agent workspace for Hermes profiles, external coding agents, task panels, and automation workflows, useful as a concrete UI reference for topic-scoped agents. https://github.com/TNJ2026/hermes-deck
- AgentsCommander coordinates multiple CLI coding agents with local file-based state, PTY support, loops, and no telemetry, a practical pattern for persistent Hermes worker swarms. https://github.com/mblua/AgentsCommander
- dorkos explores scheduling, messaging, and agent discovery for Claude Code, which is directly relevant to Hermes cron jobs and proactive agent infrastructure. https://github.com/dork-labs/dorkos
- opencode-routines ports Claude-Code-style routines to OpenCode with same-session loops, cron prompts, and host-backed standalone scheduled agents. https://github.com/EmilioEsposito/opencode-routines
- cursor-tg runs Cursor Cloud Agents from Telegram, a compact reference for mobile-first human-in-the-loop coding agent control. https://github.com/tb5z035i/cursor-tg
- DroidCtx context-builder generates Markdown infrastructure context for coding agents, a useful pattern for feeding Hermes agents accurate runtime and service maps. https://github.com/DrDroidLab/context-builder
- agentic-workflow defines a multi-agent Claude Code framework with acceptance tiers, adversary review, event ledger observability, and skill evolution ideas worth adapting to Hermes task boards. https://github.com/AgentShekel/agentic-workflow
- autonomous-agentic-research-swarm uses a repo as shared memory for operator, planner, worker, and judge agents with deterministic quality gates. https://github.com/AysajanE/autonomous-agentic-research-swarm
- DAAO provides zero-trust tunnels, remote terminals, AI agents, and live telemetry for machines, relevant to secure persistent agent infrastructure. https://github.com/daao-platform/daao
