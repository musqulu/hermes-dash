# Hermes workflow items worth opening today

- 🔥 Best: Context Infrastructure is a reference implementation for persistent coding-agent context with personal rules, skills, scheduled observations, and cron docs that map closely to Hermes memory/skill rollups. [https://github.com/grapeot/context-infrastructure](https://github.com/grapeot/context-infrastructure)
- Live Memory is a Claude Code MCP plugin that passively learns from repo reads/edits and exposes a read-only `ask_live_memory` tool, a strong pattern for cheap always-on repo memory in Hermes. [https://github.com/shofer-dev/claude-code-live-memory](https://github.com/shofer-dev/claude-code-live-memory)
- Antigravity Telegram Suite shows a mature Telegram remote-control surface for an AI IDE agent, including model switching, workspace routing, file browsing, screenshots, and multi-user controls. [https://github.com/emreturkmencom/antigravity-telegram-suite](https://github.com/emreturkmencom/antigravity-telegram-suite)
- aharness turns coding-agent workflows into enforceable state machines with typed gates, validated evidence, controlled transitions, and inspectable logs instead of relying on prompt discipline. [https://github.com/Alfredvc/aharness](https://github.com/Alfredvc/aharness)
- Agent HQ is a zero-dependency Node operating platform for multi-agent companies with shared memory, kanban, agent registry, live dashboard, SSE updates, and an MCP server. [https://github.com/tools-for-agents/agent-hq](https://github.com/tools-for-agents/agent-hq)
- Engram is a self-hosted MCP memory server and dashboard over a git-backed Markdown vault, with expiry/supersession semantics that could improve stale-memory handling. [https://github.com/rwnalds/engram](https://github.com/rwnalds/engram)
- KAOS packages a local-first durable-agent runtime with Telegram/Discord/webhook/cron entry points, memory consolidation, skills, MCP tools, automations, and a control-room dashboard. [https://github.com/spyrae/kronos-agent-os](https://github.com/spyrae/kronos-agent-os)
- Valmis is an OpenClaw-style work-agent platform with isolated dockerized agents, credential proxying, cron/webhook/app-event triggers, and per-step tool/credential restrictions. [https://github.com/valmishq/valmis](https://github.com/valmishq/valmis)
- Rowboat is a desktop AI coworker with a living knowledge graph and built-in work surfaces, useful as a reference for agent memory plus human-facing workspace UX. [https://github.com/rowboatlabs/rowboat](https://github.com/rowboatlabs/rowboat)
- InsForge is an open-source backend platform aimed at coding agents, bundling database, auth, storage, compute, hosting, and AI gateway primitives for end-to-end app shipping. [https://github.com/InsForge/InsForge](https://github.com/InsForge/InsForge)
