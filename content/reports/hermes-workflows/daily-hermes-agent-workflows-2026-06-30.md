# Hermes/agent workflows worth opening today

- 🔥 Best: Aharness turns Codex workflows into enforceable state machines with typed gates, approvals, and recovery rules for long-running coding tasks. https://github.com/Alfredvc/aharness
- CodexPro exposes a local repo workspace to ChatGPT Developer Mode through MCP, useful for turning ChatGPT into a practical coding agent without rebuilding your toolchain. https://github.com/rebel0789/codexpro
- Conduit is a local MCP gateway that lets Claude, Cursor, VS Code, Codex, and other clients share one authenticated tool setup while reducing MCP token load. https://github.com/tsouth89/conduit
- DeployBot gives coding agents a GitHub merge queue where Codex, Claude Code, Cursor, or MCP clients can prepare PRs while the human keeps the final deploy decision. https://github.com/Forward-Future/DeployBot
- Open Second Brain connects Hermes Agent to an Obsidian vault with deterministic CLI/MCP memory tools, making preferences, evidence, and audit trails editable Markdown. https://github.com/itechmeat/open-second-brain
- Janus is a local personal assistant runtime with memory, permissions, scheduled automations, Codex/LLM support, and a Telegram bridge. https://github.com/fa8i/janus-assistant
- GitHub Agentic Workflows runs natural-language agent workflows inside GitHub Actions, useful for repeatable repo maintenance and automation with guardrails. https://github.com/github/gh-aw
- AgentScan audits exposed MCP servers, A2A agent cards, and open LLM APIs in one command, useful before connecting more agents to public or local infrastructure. https://github.com/7anX/AgentScan
- GTM Skills packages 205 production go-to-market agent skills for outbound, enrichment, PLG, analytics, RevOps, founder-led sales, and marketing automation. https://github.com/LeadMagic/gtm-skills
- Hermes Atlas is a community map of Hermes Agent skills, plugins, integrations, deployment templates, and ecosystem projects worth browsing for setup ideas. https://github.com/ksimback/hermes-ecosystem
