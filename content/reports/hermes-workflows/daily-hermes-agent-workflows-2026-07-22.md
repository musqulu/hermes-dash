# Hermes/agent workflows worth opening today

- 🔥 Best: Hermes Atlas is a community map of Hermes Agent tools, skills, integrations, deployment templates, and forks, useful as the first place to scout practical building blocks. https://github.com/ksimback/hermes-ecosystem
- courier turns Telegram DMs into a durable inbox that MCP-capable coding agents can pull from later, which fits phone-to-Codex/Hermes task capture. https://github.com/ColoMiranda/courier
- codex-workflows adds repeatable Codex CLI delivery flows with planning, TDD, subagents, and quality gates for safer product-building work. https://github.com/shinpr/codex-workflows
- codex-dynamic-workflows provides scriptable agent pipelines with `agent()`, `parallel()`, `pipeline()`, budgets, and a visual run viewer across Codex, Gemini, and other backends. https://github.com/six-ddc/codex-dynamic-workflows
- Seamless is a local-first MCP memory and coordination layer so multiple coding agents can share project constraints, backlog state, and decisions without a hosted database. https://github.com/0spoon/seamless
- multimodels-mcp lets a primary coding agent delegate tasks to Codex CLI, DeepSeek, OpenRouter, GLM, or local LM Studio models through one MCP server. https://github.com/dpmadsen/multimodels-mcp
- ka-toolkit captures LLM conversations into an Obsidian-compatible Markdown knowledge base and includes MCP plus Telegram/Lark channel daemons for driving agent sessions. https://github.com/lionwashington/ka-toolkit
- OtoDock is a self-hosted server setup for running Claude Code and Codex as a coordinated team of agents using existing subscriptions. https://github.com/OtoDock/oto-dock
- Claudexor is a local-first control plane for Claude Code, Codex CLI, Cursor CLI, OpenCode, and API adapters with quota-aware routing. https://github.com/razzant/claudexor
- CTO AIPA shows a production-style small-business automation stack with PR review, web research, GTM intelligence commands, persistent memory, and Telegram access. https://github.com/ElenaRevicheva/AIPA_AITCF
