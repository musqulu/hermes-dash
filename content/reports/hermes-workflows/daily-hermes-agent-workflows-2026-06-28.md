# One thing worth opening today

Open [Agent of Empires](https://github.com/agent-of-empires/agent-of-empires): it is a practical session manager for running multiple AI coding agents from a terminal UI or browser dashboard, with git worktrees, optional Docker sandboxing, and mobile-friendly monitoring.

- Fresh signal: GitHub search showed it updated on 2026-06-27 with 2,662 stars, making it stronger than the tiny Telegram/MCP repos found today.
- Useful fit: it supports Codex CLI alongside Claude Code, Gemini CLI, OpenCode, Copilot CLI, Factory Droid Coding, and other agent runners.
- Practical workflow: use it as the operator console for parallel Hermes/Codex build loops — one agent per branch/task, visible from laptop or phone, instead of losing state across terminals.

Why it matters: Konrad can turn Hermes + Codex into a more reliable small-product build station for Analog Hive or utllo.com by supervising multiple isolated agent sessions, catching stuck agents quickly, and keeping experiments separated in worktrees.
