# Hermes workflow items worth opening today

- 🔥 Best: OtoDock is a self-hosted Claude Code/Codex team-agent server with dashboard terminals, per-agent memory, schedules, webhooks, workspaces, MCP framework, and sandbox/network isolation ideas directly relevant to Hermes gateway + cron evolution: [OtoDock](https://github.com/OtoDock/oto-dock/).
- Rowboat is a local-first work-surface app that wraps email, meeting notes, browser, notes, and parallel Claude Code/Codex orchestration around a Markdown knowledge graph: [Rowboat](https://github.com/rowboatlabs/rowboat).
- Velpos is a fresh Claude Code multi-agent management dashboard with IM collaboration, monitoring, channel configuration, message notifications, and a Codex GUI angle worth checking for Telegram/dashboard UX patterns: [velpos](https://github.com/Jxin-Cai/velpos).
- TokenTelemetry tracks tokens, sessions, tool calls, and reasoning across Hermes Agent, Claude Code, Codex, Antigravity, and other runtimes in a local dashboard: [tokentelemetry](https://github.com/VasiHemanth/tokentelemetry).
- Agent Control turns CLI coding agents into a job-board-driven team with task routing, shared memory, dashboard supervision, and human approvals: [agent-control](https://github.com/serifcolakel/agent-control).
- Mac Mini Control Room is a small web control plane for cron-driven Claude Code agents with Markdown prompts, lockfiles, log streaming, and a dashboard: [macmini-control-room](https://github.com/nicolasmertens/macmini-control-room).
- NeuroVault is a local-first MCP memory layer for Claude, Cursor, and other agents using Markdown as the canonical store with on-device retrieval: [NeuroVault](https://github.com/sirdath/NeuroVault).
- Reefiki focuses on distilling chat noise into reusable decisions, skills, and project memory, which maps well to Hermes memory rollups and skill-curation ideas: [Reefiki](https://github.com/NeAGI/Reefiki).
- Flotilla proposes a Discord-bus “fleet of AI coding agents” with a chief-of-staff coordinator, live dashboard, shared memory, and audited handoffs: [flotilla](https://github.com/jim80net/flotilla).
- Cronmaster MCP is a SQLite-backed scheduling/orchestration MCP for agent recurring jobs with dependencies, retries, logging, and alerting: [cronmaster-mcp](https://github.com/marilynceo/cronmaster-mcp).
