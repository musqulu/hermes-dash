# Hermes workflow items worth opening today

- 🔥 Best: Aharness turns Codex-style coding-agent workflows into enforced state machines with typed gates, validated evidence, repair paths, and inspectable logs — https://github.com/Alfredvc/aharness
- CCGram is a self-hosted Telegram bridge for Claude Code approvals, questions, session resumes, and agent control from a phone — https://github.com/jsayubi/ccgram
- Antigravity Telegram Suite shows a richer Telegram remote-control pattern for AI agents with chat, model switching, workspace management, screenshots, and multi-agent orchestration — https://github.com/emreturkmencom/antigravity-telegram-suite
- Stratum is a small MCP/Python state-machine dispatch server that gives coding agents typed specs, postconditions, retries, gates, and auditable traces — https://github.com/smartmemory/stratum
- Claude Agent Station is a self-hosted scheduled Claude Code team runner with manager/employee agents, GitHub task execution, and a web dashboard for live visibility — https://github.com/kenhaesler/claude-agent-station
- BagIdea Office explores a desktop “agent office” metaphor with delegated Claude Code agents, meetings, per-agent model swaps, skills, plugins, voice, and Telegram/Discord/LINE integration — https://github.com/bagidea/bagidea-office
- Fenic provides semantic DataFrames for humans and agents, useful as a structured extraction and ranking layer for recurring scouts before LLM summarization — https://github.com/typedef-ai/fenic
- Cordium is an open-source sandbox platform for devs and AI agents that could inform safer isolated runner infrastructure with identity-based access instead of raw credentials — https://github.com/octelium/cordium
- Harness Creator generates portable AI-coding harness scaffolds across Codex, Claude Code, and OpenClaw with shared workflow state, evidence, role agents, gates, and lessons — https://github.com/TheAatroxking1/harness_creator_skill
