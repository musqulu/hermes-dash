# Hermes/agent workflows worth opening today

- 🔥 Best: Bender is a self-hosted personal agent with a markdown wiki, Things-style tasks, Telegram streaming, cron memory, and self-learning skills. https://github.com/0717376/bender
- Rowboat is an open-source desktop AI coworker with work memory and built-in surfaces for acting on that memory. https://github.com/rowboatlabs/rowboat
- Agent Log Viewer turns local Codex and Claude Code logs into a live chat-style UI with session trees, subagents, and background shell tasks. https://github.com/Latand/live-log-viewer-next
- aharness makes Codex-style long-running workflows executable with typed gates, evidence checks, repair paths, and inspectable logs. https://github.com/Alfredvc/aharness
- 9lives is a self-healing Playwright test runner for coding-agent loops that can repair broken locators and show the diff. https://github.com/Quality-Max/9lives
- Dex packages analytics-engineering skills for agents so Claude Code-style workflows can inspect warehouses, dbt projects, schema drift, and transformations. https://github.com/exmergo/dex
- TypesenseKit gives agents and humans 95 typed Typesense operations through a CLI and secure MCP server with read-only access by default. https://github.com/typesensekit/typesensekit
- LlamaAgents provides event-driven async Python workflows for document-heavy agent pipelines that need extraction, validation, review, APIs, and scaling. https://github.com/run-llama/llama-agents
- joplin-cli exposes Joplin through a CLI and Python SDK for agent-driven note capture, search, and second-brain automation. https://github.com/syedmohammadzaid/joplin-cli
- Hermes Agent docs remain the canonical reference for configuring Hermes, cron jobs, Telegram delivery, skills, plugins, and Codex-connected operator workflows. https://hermes-agent.nousresearch.com/docs
