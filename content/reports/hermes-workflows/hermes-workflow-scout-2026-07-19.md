# Hermes workflow items worth opening today

- 🔥 Best: agmsg is a tiny Bash+SQLite cross-agent messaging layer for letting Claude Code, Codex, Gemini and Copilot-style CLI agents coordinate without a daemon. https://github.com/fujibee/agmsg
- TokenTelemetry now explicitly targets Hermes Agent observability, giving a local dashboard for tokens, sessions, tool calls, reasoning and multi-platform autonomous-agent runs. https://github.com/VasiHemanth/tokentelemetry
- my-claude-bot is a Telegram-first multi-agent platform with shared message bus, background memory processing, cron jobs and MCP integrations, useful as a pattern library for Hermes gateway agents. https://github.com/dream77r/my-claude-bot
- Agent Control turns CLI coding agents into a dashboard-driven job board with task routing, shared memory and human approval, mapping closely to Hermes kanban/orchestrator ideas. https://github.com/serifcolakel/agent-control
- parallel-sessions offers pure-Bash coordination for many Claude Code and Codex sessions in one repo, including file locking, crash recovery and cross-session delegation. https://github.com/sezeryavuz/parallel-sessions
- memini is a single-binary MCP memory service with SQLite/Postgres backends and hybrid vector plus keyword recall for shared persistent agent memory. https://github.com/eleboucher/memini
- SecureContext combines persistent project memory, tamper-evident audit logs, delivery state and skill verification for Claude Code, with several security patterns portable to Hermes skills/tools. https://github.com/iampantherr/SecureContext
- Watchfloor packages an eight-phase agentic SDLC for Claude Code with slash commands, specialist agents, hooks, schemas and supervision dashboard concepts worth mining for Hermes workflow templates. https://github.com/tomashermansen-lang/watchfloor
- droidctx generates Markdown infrastructure context from tools like Grafana, Datadog, Kubernetes and CloudWatch so coding agents start with grounded production context instead of pasted runbooks. https://github.com/DrDroidLab/context-builder
