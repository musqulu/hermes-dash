# Hermes workflow items worth opening today

- 🔥 Best: WakeWire pushes GitHub, Gmail, Slack, and arbitrary webhook events directly into local Codex threads, which is a strong pattern for proactive Hermes triggers without polling: [github.com/glenncalleja/wakewire](https://github.com/glenncalleja/wakewire)
- Agent Log Viewer turns Claude Code and Codex logs into a live local session/subagent/background-task tree, useful for a Hermes multi-agent dashboard: [github.com/Latand/live-log-viewer-next](https://github.com/Latand/live-log-viewer-next)
- Orbital is a persistent local project manager agent that delegates to Claude Code, Codex, Gemini, and others while keeping shared context, approvals, and budgets: [github.com/zqiren/Orbital](https://github.com/zqiren/Orbital)
- Ultracodex packages Claude-style workflows, builder-verifier loops, scheduling, and parallel Codex agent fleets behind reusable scripts: [github.com/YuanpingSong/ultracodex](https://github.com/YuanpingSong/ultracodex)
- Lich runs isolated dev stacks per git worktree so parallel coding agents can each get their own ports, containers, env, and lifecycle: [github.com/RPate97/lich](https://github.com/RPate97/lich)
- Skillful stores reusable AI skills and agents as local folders and installs them across Claude Code, Codex, Cursor, Copilot, Gemini, Junie, and opencode: [skillful.md](https://skillful.md/)
- Twill’s cloud-agent PR workflow emphasizes real-stack execution plus proof artifacts like test output, screenshots, and live previews that Hermes coding jobs could imitate: [twill.ai](https://twill.ai)
- Cosmic Team Agents show a concrete Slack, WhatsApp, and Telegram agent pattern for operating a CMS from team chats, relevant to Hermes gateway action design: [cosmicjs.com/blog/introducing-team-agents-and-cosmic-agent](https://www.cosmicjs.com/blog/introducing-team-agents-and-cosmic-agent)
