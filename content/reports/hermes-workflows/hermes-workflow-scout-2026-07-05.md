# Hermes workflow items worth opening today

- 🔥 Best: Tokentelemetry is a local dashboard for tracking tokens, sessions, tool calls, and reasoning across Hermes Agent, Claude Code, Codex, Antigravity CLI, and similar agents: https://github.com/VasiHemanth/tokentelemetry
- Takuto Core shows a ticket-to-PR pipeline that polls Jira/GitHub, launches headless coding agents in isolated worktrees, and exposes progress through a live dashboard: https://github.com/takuto-team/takuto-core
- Mac Mini Control Room is a small web control plane for cron-driven Claude Code agents with prompt-as-Markdown, lockfiles, log streaming, and a local dashboard: https://github.com/nicolasmertens/macmini-control-room
- Agentic Task System turns existing task managers into persistent MCP-backed agent memory with hybrid retrieval and adapters for tools like TickTick, Obsidian, Notion, GitHub, Airtable, Taskmaster, and Beads: https://github.com/renezander030/agentic-task-system
- AI Knowledge Base is a GitHub Trending/Hacker News/RSS ingestion workflow that uses an LLM to structure AI-agent updates and push them to Telegram or Feishu: https://github.com/quarktimes/ai-knowledge-base
- Hamroh is a self-owned Telegram persistent companion harness with an extendable custom-agent template and learning-oriented architecture: https://github.com/Rustam-Z/hamroh
- Telegram Agent is a serverless Bun-on-Cloud-Run personal assistant pattern for Telegram with scale-to-zero hosting, GitHub-backed memory, and swappable LLM providers: https://github.com/cherrythia/telegram-agent
- Captain Claw is a self-hosted model-agnostic framework for orchestrating specialist AI-agent fleets with ensemble reasoning and an agentic coding pipeline: https://github.com/kstevica/captain-claw
- Gortex is a local code-intelligence graph engine with CLI, MCP, and API access that can reduce coding-agent context load across large multi-repo workspaces: https://github.com/zzet/gortex
- Kmux is a keyboard-centric terminal workspace for running multiple coding agents side by side with session resume, usage dashboards, and automated git worktrees: https://github.com/kkd927/kmux
