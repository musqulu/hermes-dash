# Hermes/agent workflows worth opening today

- 🔥 Best: Hermes Atlas is a community map of Hermes Agent tools, skills, integrations, deployment templates, and forks, useful as the first place to scout installable leverage. Source: https://github.com/ksimback/hermes-ecosystem
- Hermes Evolution shows a self-improvement loop for Hermes where the agent researches upgrades, proposes changes, and applies daily updates. Source: https://github.com/Lexus2016/hermes-agent-evolution
- Hermes Agent Automations documents a concrete VPS setup with separate CEO and CTO Hermes agents, Telegram gateways, cron jobs, calendar scanning, code review, and tool discovery. Source: https://github.com/geraledesma/hermes-agent-automations
- Echo Agent is a self-hosted long-running personal/team agent with memory, approval boundaries, MCP/A2A support, scheduled tasks, and Telegram or WeChat entry points. Source: https://github.com/fuyuxiang/echo-agent
- OtoDock packages Claude Code and Codex as collaborative self-hosted agents on your own server using existing subscriptions and shared team tooling. Source: https://github.com/OtoDock/oto-dock
- Ciaobot is a local second-brain web app that wraps Claude Code or OpenAI Codex around projects, markdown vaults, schedules, memory, and Google Workspace workflows. Source: https://github.com/raffaelefarinaro/ciaobot
- Agentic Awesome Skills is a large installable library of reusable SKILL.md playbooks and workflow bundles for Codex CLI, Claude Code, Cursor, Gemini CLI, and other coding agents. Source: https://github.com/sickn33/agentic-awesome-skills
- Fluxito is an MCP-enabled marketing analytics system for planning, implementing, auditing, reporting, and automating analytics work across many marketing platforms. Source: https://github.com/digitalXperiments/fluxito
- Octopus LinkedIn is an MCP server and CLI for governed LinkedIn drafting, approval, publishing, commenting, and engagement reading through the official LinkedIn API. Source: https://github.com/octoryn/octopus-linkedin
- Vexa is a self-hostable meeting bot and transcription API with an MCP server that can feed agent workflows from Google Meet, Teams, Zoom, and Jitsi calls. Source: https://github.com/Vexa-ai/vexa
