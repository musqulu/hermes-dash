# Hermes workflow items worth opening today

- 🔥 Best: 143.dev is a newly open-sourced team coding-agent platform for running Codex, Claude Code, OpenCode, and similar agents in isolated gVisor sandboxes with shared integrations and introspection: https://github.com/assembledhq/143
- Aegize is a small Python runtime layer for deterministic tool-call policy, approvals, and append-only audit logs around autonomous agents: https://github.com/gggaswint/aegize
- OpenTag is a Slack-native gateway pattern for routing team threads to Claude Code, Codex, OpenCode, Docker, HTTP agents, or custom CLIs with policy, memory, approvals, artifacts, and audit logs: https://github.com/linxidnju/OpenTag
- Helio is an MCP governance proxy that could inform Hermes-side tool policy, approval routing, evidence checks, spend tracking, and per-tool audit without changing agent code: https://github.com/gethelio/helio
- Aharness shows a Codex workflow harness with typed gates, validated evidence, controlled transitions, repair paths, and inspectable logs for stricter autonomous coding runs: https://github.com/Alfredvc/aharness
- Circus Chief is a browser/phone control plane for managing Claude Code, OpenAI Codex, and Gemini CLI agents, useful as a reference for remote mobile supervision of coding agents: https://github.com/ferrislucas/Circus-Chief
- Antigravity Telegram Suite is a Telegram remote-control bot for chatting with an AI agent, switching models, orchestrating multi-agent workflows, and managing workspaces from mobile: https://github.com/emreturkmencom/antigravity-telegram-suite
- Phleet combines Dockerized Claude/Codex agents, Temporal orchestration, and Telegram coordination, making it a useful reference for durable multi-agent jobs beyond a single chat session: https://github.com/anurmatov/phleet
- Legax is a mobile-first, self-hosted relay for remote-controlling coding agents with Telegram support and local-first privacy constraints: https://github.com/zhanex/legax
- AgentPass is a stateful guardrail layer for tool calls with job state, approvals, idempotency, PII egress controls, budgets, and audit before execution: https://github.com/dinpd/AgentPass
