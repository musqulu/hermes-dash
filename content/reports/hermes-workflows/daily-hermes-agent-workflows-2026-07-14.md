# Hermes/agent workflows worth opening today

- 🔥 Best: ultracodex runs Claude Code-style workflow scripts on OpenAI Codex/OpenCode, with agent loops, schedulers, verifier gates, and persistent agent “orgs” for cheaper multi-agent coding work. https://github.com/YuanpingSong/ultracodex
- n8n-nodes-ai-cli lets n8n workflows call locally authenticated Claude Code and OpenAI Codex CLIs, useful for turning subscription-based coding agents into automations without API keys. https://github.com/riddhimaaan/n8n-nodes-ai-cli
- Alfi is a public Hermes Agent personal-assistant architecture using Telegram, a Markdown wiki, Hermes cron, deterministic scripts, and optional Codex workers. https://github.com/MatusNebus/alfi-personal-assistant-system
- Galahad is a self-hosted Telegram-controlled team of autonomous agents for a VPS, with orchestrator, health patrol, explorer, and optional Claude bridge roles. https://github.com/xtincell/galahad
- Fleet Deck gives local mission control for many Claude Code sessions, showing live status, permission waits, conflicts, and captured plans from one board. https://github.com/lacion/fleet-deck
- TeleBoard turns Telegram group chaos into Jira/OpenProject tickets with CrewAI classification, ticket updates, approver DMs, SLA follow-ups, and reply drafts. https://github.com/raghupviyer/TeleBoard
- Mnema is a self-hostable MCP knowledge workspace that publishes docs, decisions, and workflows to Claude, ChatGPT, Cursor, Windsurf, and other MCP-aware agents. https://github.com/nbkdoesntknowcoding/mnema
- humanforai-mcp gives agents an MCP tool for hiring a real human operator for app testing, product verification, design judgment, data collection, or physical-world checks. https://github.com/humanforai/humanforai-mcp
- AI-Marketing-Workflow collects reusable Claude Code-style marketing skills for growth experiments, SEO, content ops, outbound, conversion audits, sales pipeline work, and revenue reporting. https://github.com/TruongThuHa/AI-Marketing-Workflow
- second-brain-setup is a single-file memory system for Claude Code, Codex CLI, and other agents that keeps decisions, context, and reusable knowledge across sessions. https://github.com/Alxve99/second-brain-setup
