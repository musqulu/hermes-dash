# Hermes/agent workflows worth opening today

- 🔥 Best: cc-connect is a practical bridge for running local coding agents like Codex, Claude Code, Cursor, or Gemini from Telegram, Slack, Discord, LINE, Lark, DingTalk, and WeChat Work without exposing your machine publicly. https://github.com/chenhg5/cc-connect
- claw-skeleton is a compact open example of a Telegram-native personal assistant where every chat topic maps to an isolated coding-agent session with persistent memory and proactive cron heartbeats. https://github.com/LuKresXD/claw-skeleton
- kai shows a security-minded Telegram coding assistant pattern with persistent backends, local shell access, PR review automation, scheduled jobs, and explicit control boundaries. https://github.com/dcellison/kai
- Activepieces is worth mining for small-business leverage because it combines AI agents, workflow automation, and hundreds of MCP-style integrations for marketing, ops, and SaaS glue work. https://github.com/activepieces/activepieces
- Google’s MCP Toolbox for Databases is a strong building block for letting an agent safely query and operate on product, analytics, or CRM databases through a dedicated MCP server. https://github.com/googleapis/mcp-toolbox
- Figma Context MCP gives coding agents structured Figma layout context, useful for turning Analog Hive or utllo.com design work into more accurate frontend implementation tasks. https://github.com/GLips/Figma-Context-MCP
- Worktrunk targets parallel AI-agent development by managing Git worktrees, which is useful when you want multiple Codex-style agents to explore fixes or product ideas without trampling each other. https://github.com/max-sixty/worktrunk
- GitHub Agentic Workflows is GitHub’s own experiment around agent-run development flows, worth tracking for repo-native automation and issue-to-PR workflows. https://github.com/github/gh-aw
- agent-infra/sandbox packages browser, shell, file access, MCP, and VS Code Server into one Docker sandbox, which is useful for safer autonomous research or build agents. https://github.com/agent-infra/sandbox
- Headroom compresses tool outputs, logs, files, and RAG chunks before they hit the model, making long-running agent workflows cheaper and less likely to drown in noisy context. https://github.com/headroomlabs-ai/headroom
