# Hermes workflow items worth opening today

- 🔥 Best: OtoDock is a fresh self-hosted control plane for running Claude Code and Codex agents as a team on your server, directly relevant to Hermes multi-agent dashboards and BYO-subscription orchestration. https://github.com/OtoDock/oto-dock
- TokenTelemetry is a local dashboard for tracking tokens, sessions, tool calls, and reasoning across Hermes Agent, Claude Code, Codex, and similar agents. https://github.com/VasiHemanth/tokentelemetry
- claw-skeleton is an open Telegram-first assistant skeleton where each topic gets an isolated Claude Code agent with layered memory and proactive cron heartbeats. https://github.com/LuKresXD/claw-skeleton
- macmini-control-room shows a simple web control plane for autonomous coding agents on a Mac Mini with prompt-as-Markdown jobs, lockfiles, cron, logs, and a dashboard. https://github.com/nicolasmertens/macmini-control-room
- session-briefing offers a lightweight pattern for carrying coding-agent project state across sessions with timestamped briefings plus cross-project rollups. https://github.com/CaptCanadaMan/session-briefing
- Papai is a proactive personal-agent repo focused on scheduled nudges, smart suggestions, and context-aware assistance that could inform Hermes cron notification UX. https://github.com/yourpapai/papai
- Watchfloor packages an eight-phase autonomous SDLC pipeline for Claude Code with orchestration and a supervision dashboard. https://github.com/tomashermansen-lang/watchfloor
- Superharness is a multi-agent orchestration harness for Claude Code plus Codex with task lifecycle, handoff protocol, dashboard, and autonomous dispatch ideas. https://github.com/artificemachine/superharness
- Creel explores self-hosted topic-scoped memory-as-a-service for agents with RBAC, linked chunks, and dual-mode retrieval. https://github.com/Tight-Line/creel
- frontend-visualqa gives coding agents a visual QA loop for verifying frontend work, useful as a pattern for Hermes browser/vision-based completion checks. https://github.com/yutori-ai/frontend-visualqa
