# Hermes workflow items worth opening today

- 🔥 Best: Pulse is a Claude Code dashboard pattern for remote tool-call approvals from a phone, directly relevant to Telegram/mobile command approval UX: [github.com/nikitadoudikov/claude-pulse](https://github.com/nikitadoudikov/claude-pulse)
- Takuto Core shows a ticket-to-PR pipeline that polls Jira/GitHub, runs headless coding agents in isolated worktrees, and exposes progress in a live dashboard: [github.com/takuto-team/takuto-core](https://github.com/takuto-team/takuto-core)
- Microsoft Dev-Trio packages a Planner → Implementer → Critic loop for Copilot Chat with notifications, project memory, and a dashboard: [github.com/microsoft/Dev-Trio](https://github.com/microsoft/Dev-Trio)
- Hamroh is a self-owned persistent Telegram agent harness with an extendable companion template and learning/memory emphasis: [github.com/Rustam-Z/hamroh](https://github.com/Rustam-Z/hamroh)
- Semantic Daily demonstrates a practical multi-source scouting pipeline across Reddit, Bluesky, GitHub, RSS, Hacker News, YouTube, and Exa with Telegram and dashboard delivery: [github.com/ashok-kallara/semantic-daily](https://github.com/ashok-kallara/semantic-daily)
- Agentic Task System turns existing task managers into persistent MCP-backed agent memory with hybrid retrieval and adapters for TickTick, Obsidian, Notion, GitHub, Airtable, Google, OKF, Taskmaster, and Beads: [github.com/renezander030/agentic-task-system](https://github.com/renezander030/agentic-task-system)
- Brain.md is a small persistent-memory layer for coding agents that is worth mining for lightweight Markdown memory conventions: [github.com/mindmuxai/brain.md](https://github.com/mindmuxai/brain.md)
- Orloj frames agent infrastructure as YAML/GitOps, a useful angle for versioned Hermes cron, profile, and workflow deployment recipes: [github.com/OrlojHQ/orloj](https://github.com/OrlojHQ/orloj)
- Claude Agent Station uses a manager/employee architecture plus a web dashboard for multi-project GitHub automation with Claude Code: [github.com/kenhaesler/claude-agent-station](https://github.com/kenhaesler/claude-agent-station)
- AI-company collects Claude Code-native multi-agent team patterns with lifecycle hooks, pipeline workflows, persistent teams, a task wall, and a React dashboard: [github.com/CronusL-1141/AI-company](https://github.com/CronusL-1141/AI-company)
