# Hermes/agent workflows worth opening today

- 🔥 Best: Hermes Agent v2026.7.1 adds `/learn` for reusable skills, verification evidence for coding work, background subagent fan-out, stronger Telegram cron delivery, and desktop coding Projects.  
  Source: https://github.com/NousResearch/hermes-agent/releases/tag/v2026.7.1
  - Evidence: released July 1; highlights include `/learn`, completion contracts, verification ledger, background delegation, Telegram command-menu fixes, and project tooling.

- OpenTag turns Slack/GitHub/Lark `@agent` mentions into routed work for local Codex, Claude Code, or similar coding agents, then returns results in the thread.  
  Source: https://github.com/amplifthq/opentag

- Agent Apprenticeship packages long-horizon agent workflow loops, reusable lessons, seed tasks, and execution traces for making repeated operator work compound.  
  Source: https://github.com/Forsy-AI/agent-apprenticeship

- UmaDev wraps Claude Code, Codex, or OpenCode as a small AI dev team with PM, architect, UI/UX, frontend, backend, QA, security, DevOps, and coordinator roles.  
  Source: https://github.com/umacloud/umadev

- Kiro Telegram Bot is a mobile control layer for Kiro CLI over ACP, with project switching, live coding sessions, streamed diffs, queued follow-ups, and 24/7 service mode.  
  Source: https://github.com/artickc/kiro-telegram-bot

- herdr-remote adds phone, menu-bar, web, and Telegram approval flows for supervising blocked agents without SSH.  
  Source: https://github.com/dcolinmorgan/herdr-remote

- auto-vlog-editor turns daily phone clips into narrated vertical videos, sends the result to Telegram for approval, and can auto-publish after review.  
  Source: https://github.com/tstambulsky/auto-vlog-editor

- OmniRAG-n8n is a self-hosted n8n workflow that ingests Google Drive docs into Pinecone with local Ollama embeddings and answers through a Telegram AI agent.  
  Source: https://github.com/ziadnasar6-lgtm/OmniRAG-n8n

- gemini-search-mcp gives agents a free no-key MCP web-search server backed by Google AI Mode, useful when Hermes or Codex needs fresh research context.  
  Source: https://github.com/Sophomoresty/gemini-search-mcp

- Hermes Skills is a small community library of plug-and-play Hermes skills for SEO, WordPress, marketing, finance, and web workflows.  
  Source: https://github.com/RobinBeraud/hermes-skills
